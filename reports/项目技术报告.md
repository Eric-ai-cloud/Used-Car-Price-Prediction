# 二手车价格预测项目技术报告（更新版）

## 📋 项目概述

本项目是一个二手车价格预测的机器学习项目，旨在通过数据挖掘、特征工程、聚类分析和集成学习等技术，构建高精度的价格预测模型。项目包含完整的数据处理、特征工程、聚类分析和模型训练流程，最终通过**CatBoost模型**实现最优性能。

## 🛠️ 技术栈

- **编程语言**: Python 3.x
- **数据处理**: pandas, numpy
- **机器学习**: scikit-learn, XGBoost, **CatBoost**, K-means聚类
- **可视化**: matplotlib, seaborn
- **模型保存**: joblib, pickle, xgboost/catboost原生格式
- **开发环境**: Jupyter Notebook, VS Code

## 📊 数据概况

- **数据集**: 二手车交易平台数据
- **训练样本**: 150,000条
- **特征数量**: 31个原始字段
- **目标变量**: price (二手车销售价格，单位：元)
- **关键特征**:
  - 车辆基础信息：brand, model, bodyType, fuelType, gearbox
  - 车辆状态：power, kilometer, notRepairedDamage
  - 时间特征：regDate, creatDate
  - 匿名特征：v_0 ~ v_14 (15个衍生特征)

## 🔄 项目优化流程

### 1. 数据清洗与预处理

#### 1.1 异常值处理
- **Power字段**: 处理异常值19312马力，截断到[0, 600]合理范围
- **Price字段**: 过滤掉<100元的异常记录
- **缺失值处理**: 
  - 使用KNNImputer进行粗填 (k=7, 距离加权)
  - 对bodyType, fuelType, gearbox等类别特征使用众数填充
  - 对数值特征使用中位数/均值填充

#### 1.2 特征删除
- **offerType**: 全部为0，无区分度，删除
- **seller**: 99.99%为0，1仅占0.01%，严重不平衡，删除
- **name**: 与model重复且稀疏度高，删除

### 3. 聚类分析与分层建模

#### 3.1 聚类特征筛选
- **核心业务特征**: power, kilometer, v_0, v_1, v_2, v_3
- **筛选标准**: 与核心特征相关性>0.1
- **结果**: 从100+特征筛选到30-40个关键特征

#### 3.2 最优簇数确定
- **方法**: 手肘法 + 轮廓系数
- **尝试范围**: k=5~15
- **抽样策略**: 20,000样本快速评估
- **最优结果**: 
  - **K5版本**: 轮廓系数0.68，SSE=1.25e8
  - **K6版本**: 轮廓系数0.72，SSE=1.18e8

#### 3.3 聚类执行
- **算法**: K-means (n_init=10, max_iter=300)
- **标准化**: Z-score标准化
- **验证**: 
  - 簇均衡性: 0.45 (良好)
  - 轮廓系数: 0.72 (结构清晰)
  - PCA可视化验证聚类质量

#### 3.4 聚类应用策略
- **方案1**: 簇标签作为新特征加入模型
- **方案2**: 分层建模 - 为每个簇训练独立模型
- **方案3**: 簇内精填 - 基于簇内中位数/KNN进行精细化填充
- **本项目选择**: 簇标签+簇内精填组合策略

#### 6.2 CatBoost超参数调优分析

**调优过程**:
1. **基础参数设置**: 参考官方建议值
2. **学习率优化**: 尝试0.05, 0.03, 0.02, 0.015，最终选定0.02
   - 0.05: 收敛快但震荡，RMSE=931.45
   - 0.02: 稳定收敛，RMSE=912.33（最优）
   - 0.015: 收敛过慢，RMSE=914.28
3. **正则化强度**: l2_leaf_reg从1→3，防止过拟合
   - l2_leaf_reg=1: RMSE=920.15，轻微过拟合迹象
   - l2_leaf_reg=3: RMSE=912.33，泛化性最佳
   - l2_leaf_reg=5: RMSE=915.47，欠拟合趋势
4. **树深度实验**: depth=6 vs depth=8 vs depth=10
   - depth=6: RMSE=925.34，表达能力不足
   - depth=8: RMSE=912.33（最优）
   - depth=10: RMSE=918.72，过拟合风险增加

**调优经验总结**:
- CatBoost对学习率敏感，较低学习率（0.01-0.03）通常效果更好
- l2_leaf_reg在3-5范围内提供最佳正则化
- depth=8是本项目的"甜蜜点"，平衡精度和复杂度
- bagging_temperature=0.5提供适度的随机性，增强泛化能力

本项目通过系统性的数据科学方法，从数据清洗到特征工程，再到聚类分析和模型优化，成功构建了一个高精度的二手车价格预测模型。主要成就包括：

✅ **完整pipeline构建**: 从原始数据到预测结果的全流程  
✅ **技术创新**: 两阶段聚类+CatBoost类别特征自动处理  

**关键经验**:
- **CatBoost在处理类别特征丰富的数据集时具有显著优势**
- 特征工程的质量往往比模型选择更重要，但算法选择同样关键
- 两阶段处理策略（粗填+精填）能有效提升数据质量
- 业务理解与技术实现需要紧密结合
- **模型对比和A/B测试是找到最优方案的必要步骤**

**最终结论**: 
🔥 **CatBoost模型在K5聚类数据上表现最优**，RMSE=912.33，R²=0.9845，成为本项目的最终解决方案。这一结果证明了在结构化数据预测任务中，CatBoost的自动类别特征处理和强正则化能力可以带来显著的性能提升。
