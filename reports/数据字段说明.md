# 二手车价格预测数据集 - 字段说明文档

## 📊 数据集概览

- **数据来源**: 二手车交易平台

## 🔑 字段详细说明

### 1. 基础信息字段

| 字段名 | 数据类型 | 含义 | 示例/备注 |
|--------|---------|------|-----------|
| **SaleID** | int64 | 销售订单唯一标识符 | 从 0 到 149999 |
| **name** | int64 | 车辆名称编码（匿名化处理） | 已脱敏的数字编码 |

### 4. 地区与销售信息字段

### 异常值检测

1. **power 字段**:
   - 最大值为 19312 马力，明显异常
   - 建议：删除 power > 500 的异常记录

2. **price 字段**:

## 💡 特征工程建议

### 1. 时间特征提取
```python
# 从 regDate 和 creatDate 提取
- 车龄 = 交易年份 - 注册年份
- 交易月份、星期
- 是否节假日
```

### 2. 功率相关特征
```python
- 功率/里程比
- 单位里程价格
```

### 3. 分类特征编码
```python
- brand, model, bodyType, fuelType 使用 Target Encoding 或 Frequency Encoding
- regionCode 可以考虑分组或聚类
```

### 4. 需要删除的特征
```python
- offerType (无变化)
- seller (严重不平衡)
- name (与model重复，且稀疏度高)
```

## 🎯 建模注意事项

4. **特征缩放**: 对数值型特征进行标准化/归一化
5. **交叉验证**: 使用时间序列交叉验证避免数据泄露
